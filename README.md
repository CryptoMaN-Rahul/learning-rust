# learning-rust

Me learning Rust by building real tools — in public.

**The rule:** I write 100% of the code myself. No AI-generated code goes into these projects. AI agents help me *understand* — concepts, the borrow checker, errors, docs, code review — but never write it for me. This repo exists partly to break the copy-paste-from-AI habit and actually build the skill.

## Layout
- Each project is a subdirectory (`extract/`, …), its own crate.
- [`ROADMAP.md`](ROADMAP.md) — what I'm building, in order, with progress.
- AI agents working locally in this repo are bound by a strict contract (local file, not committed): they explain and review, they never write the code.

Started July 2026.
